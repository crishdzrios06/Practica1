# Práctica 1: Operaciones Básicas sobre Lenguajes

Aplicación con interfaz gráfica desarrollada en Python utilizando Flet para realizar operaciones básicas sobre lenguajes formales.

## Descripción

Este programa permite realizar diversas operaciones relacionadas con la teoría de lenguajes formales:

### Operaciones sobre cadenas

* Cálculo de **prefijos**
* Cálculo de **sufijos**
* Cálculo de **subcadenas**
* Posibilidad de **guardar los resultados en un archivo de texto**

### Operaciones sobre alfabetos

* Cálculo de la **cerradura de Kleene (Σ*)**
* Cálculo de la **cerradura positiva (Σ+)**
* Generación de cadenas a partir de un alfabeto dado y una longitud máxima.

La aplicación cuenta con una interfaz gráfica interactiva que permite al usuario ingresar los datos y visualizar los resultados de forma clara.

---

## Requisitos

Antes de ejecutar el programa es necesario tener instalado:

* Python 3.x
* La biblioteca **Flet**

---

## Instalación

1. Clonar el repositorio:

```
git clone https://github.com/usuario/nombre-repositorio.git
```

2. Entrar a la carpeta del proyecto:

```
cd nombre-repositorio
```

3. Instalar la dependencia necesaria:

```
pip install flet
```

---

## Ejecución del programa

Para ejecutar la aplicación, utilizar el siguiente comando en la terminal:

```
python app.py
```

Esto abrirá la interfaz gráfica en el navegador donde se podrán realizar las operaciones solicitadas.

---

## Archivo de resultados

El programa incluye la opción de **guardar los resultados en un archivo de texto** llamado:

```
resultados.txt
```

Este archivo se genera automáticamente en la carpeta del proyecto.
